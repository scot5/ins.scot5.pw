Server Features

Ultra-Brutal!

The server config has been edited to make all maps nearly impossible to win!
Perfect teamwork and skilled players are a absolute minimum.

  • 40 bots set to brutal difficulty
  • Bots respawn almost instantly
  • Bots are more perceptive
  • Bots fire RPGs faster than normal
  • Capping objectives takes 90 seconds instead of 30 seconds
  • Full friendly damage
  • Cashes take more damage and barely decrease bot count
  • And much more...

Map selection

Because of the Administrators personal hate against night maps and peak, these maps have been disabled and are not playable.
Custom maps have not been added to give a experience as close to vanilla as possible.

Auto kick

To ensure the best experience to the players on this server players with a ping higher than 200 will be kicked automatically.
Admins may kick you for a high ping even when your ping is not that high. Thank you for understanding.

Original server configs

The server config used is based on the config publicly available at https://github.com/lech/Conq40.

This server has been installed to allow Europeans to play on the same difficulty while enjoying a better connection.
